# FinFlow - Quick Start Guide 🚀

Welcome to **FinFlow**, your next-generation Gen-Z financial platform!

## 🎯 What You're Looking At

A complete, production-ready mobile app UI/UX featuring:

### ✨ Key Features
- **Smart Spending Tracker** - Know exactly what you can spend without guilt
- **AI Finance Coach** - Get personalized advice in Gen-Z language
- **Investment Simulator** - Practice crypto, stocks, forex & SIP with zero risk
- **Gamification** - Levels, XP, streaks, and achievements
- **Budget Management** - Visual category tracking with AI suggestions
- **Savings Goals** - Multiple goals with progress tracking
- **Analytics Dashboard** - Beautiful charts and insights

## 🎨 Design Highlights

- **Dark Mode First** - Deep space blue background (#0B0F1A)
- **Glassmorphism** - Frosted glass cards with backdrop blur
- **Gradient Magic** - Purple → Blue primary gradient (#7F5AF0 → #3DA9FC)
- **Neon Accents** - Pink (#FF4D8D) and Green (#2CB67D)
- **Smooth Animations** - Motion-powered transitions
- **Mobile-First** - Optimized for iPhone 13+ screens

## 📱 Navigation

### Starting Points:
1. **Login** (`/`) - Beautiful auth screen with social login
2. **Onboarding** (`/onboarding`) - 3-step personalized setup
3. **Home** (`/app`) - Main dashboard after login

### Bottom Nav (5 Tabs):
- 🏠 **Home** - Dashboard with balance & quick actions
- 📊 **Budget** - Category-wise spending limits
- 🎯 **Goals** - Savings goals tracker
- 🎮 **Simulate** - Investment practice lab
- 👤 **Profile** - User stats & settings

### Additional Pages:
- 💬 **AI Coach** - Chat with your financial buddy
- 📈 **Insights** - Analytics and patterns
- ➕ **Add Expense** - Quick expense input

## 🎮 Gamification System

Track your progress through:
- **Levels** (Current: Level 12)
- **XP Points** (850/1000 to next level)
- **Daily Streaks** (7 days 🔥)
- **Achievements** (24 badges earned)

## 🛠️ Tech Stack

Built with modern web technologies:
- React 18 + TypeScript
- React Router v7 (Data mode)
- Tailwind CSS v4
- Motion (Framer Motion)
- Recharts for analytics
- Lucide React icons

## 🎨 Color Palette

```css
Background: #0B0F1A (Deep Space)
Primary: #7F5AF0 → #3DA9FC (Purple → Blue)
Success: #2CB67D (Green)
Warning: #FF4D8D (Pink)
Text: #FFFFFF with opacity variants
```
